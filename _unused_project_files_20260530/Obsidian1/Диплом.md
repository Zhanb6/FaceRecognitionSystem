# Диплом

## Тема

Разработка системы распознавания лиц с веб-интерфейсом администрирования.

## Актуальность

Системы распознавания лиц используются для автоматизации контроля доступа, повышения безопасности и удобного мониторинга событий. Такие системы позволяют быстро идентифицировать пользователей, хранить историю распознаваний и управлять доступом через веб-интерфейс.

## Цель проекта

Цель проекта — разработать веб-систему для управления пользователями, камерами и распознаванием лиц на основе FaceNet.

## Задачи проекта

- изучить методы распознавания лиц
- выбрать технологии для backend и frontend
- реализовать backend API
- реализовать frontend interface
- добавить регистрацию и авторизацию пользователей
- реализовать управление пользователями и камерами
- подключить FaceNet для распознавания лиц
- реализовать журнал распознавания
- подготовить проект к запуску через Docker

## Объект исследования

Процесс автоматизированной идентификации пользователей по изображению лица.

## Предмет исследования

Методы и программные средства для реализации системы распознавания лиц с веб-интерфейсом.

## Используемые технологии

- Python
- FastAPI
- React
- TypeScript
- SQLite
- Docker
- FaceNet

## Ожидаемый результат

В результате выполнения проекта должна быть создана система, которая позволяет:

- регистрировать пользователей
- добавлять Face ID
- создавать камеры
- выполнять распознавание лиц
- сохранять историю распознавания
- управлять системой через веб-интерфейс
