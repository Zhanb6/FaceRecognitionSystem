# Архитектура проекта

Проект DiplomaProjectAITU — это система администрирования распознавания лиц.

## Основные части системы

- Frontend: React + Vite
- Backend: FastAPI
- База данных: SQLite
- AI-модуль: FaceNet
- Запуск: Docker Compose

## Как работает система

1. Пользователь открывает веб-интерфейс.
2. Frontend отправляет запросы на Backend API.
3. Backend проверяет JWT-токен пользователя.
4. Данные сохраняются в SQLite.
5. FaceNet используется для регистрации и проверки лиц.
6. Результаты распознавания сохраняются в журнал логов.
