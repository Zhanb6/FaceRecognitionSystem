# Backend FastAPI

Backend находится в папке:

`face_recognition_system/app`

## Основные файлы

- `main.py` — запуск FastAPI-приложения
- `database.py` — подключение к базе данных
- `models.py` — таблицы базы данных
- `schemas.py` — схемы запросов и ответов
- `auth.py` — авторизация, JWT, пароли
- `routers/` — API endpoints

## Основные функции backend

- регистрация пользователей
- вход в систему
- JWT-авторизация
- управление пользователями
- управление Face ID
- управление камерами
- хранение логов распознавания
