# База данных

В проекте используется SQLite.

## Файл базы данных

`face_recognition_system/db.sqlite3`

## Подключение

Подключение к базе описано в файле:

`face_recognition_system/app/database.py`

## Основные таблицы

### Users

Хранит пользователей системы.

Поля:

- id
- username
- email
- password hash
- role
- company
- is_staff
- is_camera

### Companies

Хранит компании/организации.

Поля:

- id
- name

### Face Profiles

Хранит профили лиц пользователей.

Используется для управления Face ID и доступом.

### Recognition Logs

Хранит историю распознавания.

Пример данных:

- кто был распознан
- какая камера использовалась
- время события
- точность распознавания

### Audit Logs

Хранит действия администраторов.

Например:

- создание пользователя
- изменение данных
- удаление записей

## Зачем нужна база данных

База данных нужна для хранения:

- пользователей
- ролей
- компаний
- камер
- профилей лиц
- логов распознавания
- истории действий администраторов
