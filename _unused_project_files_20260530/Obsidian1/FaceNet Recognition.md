# FaceNet Recognition

FaceNet используется для распознавания лиц в проекте.

## Назначение

FaceNet нужен для:

- получения признаков лица из изображения
- сохранения Face ID пользователя
- сравнения нового изображения с сохранёнными лицами
- определения совпадения
- расчёта точности распознавания

## Основные файлы

- `facenet_recognition.py` — основной модуль распознавания
- `face_recognition_system/facenet_recognition.py` — версия модуля внутри backend-пакета
- `face_recognition_system/app/facenet_bridge.py` — связь FastAPI с FaceNet
- `facenet_database.pkl` — файл с сохранёнными embeddings лиц

## Как работает

1. Пользователь добавляет фото для Face ID.
2. Backend получает изображение.
3. FaceNet обрабатывает лицо и создаёт embedding.
4. Embedding сохраняется в базе FaceNet.
5. При проверке новое фото сравнивается с сохранёнными embeddings.
6. Если совпадение выше порога, система определяет пользователя.

## Где используется

- добавление Face ID к пользователю
- проверка лица через камеру
- создание логов распознавания
